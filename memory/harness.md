# Operating Harness

Repeatable workflows for running this workspace with Codex and Claude.

## Session Start

1. Read `AGENTS.md`.
2. Inspect relevant `memory/`, `data/`, or `tools/` files.
3. Clarify the current objective and expected output.
4. Separate facts, assumptions, and open questions.

## Strategy Review

1. Summarize current objective.
2. Identify constraints and available options.
3. Evaluate tradeoffs.
4. Recommend a decision or next experiment.
5. Record decisions in `memory/decisions.md`.

## Anxiety-To-Focus Review

Use this when the user feels entrepreneurial anxiety, lack of momentum, or vague inability to focus.

1. Name the anxiety precisely before solving it.
2. Separate:
   - fear to discard,
   - fear to convert into focus,
   - ambition to preserve.
3. Identify whether the missing asset is execution, information, a strategic map, or emotional recovery.
4. If the missing asset is a strategic map, choose one active project as the focus object.
5. Convert the anxiety into one concrete study / strategy question.
6. End with a 60-90 minute work block that produces a written claim, product question, or next experiment.
7. Record durable insights in the most specific `memory/` file and avoid over-generalizing a single emotional state.

## Project Review

1. Update `memory/projects.md`.
2. Identify blocked, stale, or high-risk work.
3. Define the next concrete action for each active project.
4. Move raw notes from `memory/inbox.md` into durable files.
