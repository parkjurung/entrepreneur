# Operating Harness

Repeatable workflows for running this workspace with Codex and Claude.

## Session Start

1. Read `AGENTS.md`.
2. Inspect relevant `memory/`, `data/`, or `tools/` files.
3. Clarify the current objective and expected output.
4. Separate facts, assumptions, and open questions.

## Strategy Review

1. Summarize current objective.
2. Identify constraints and available options.
3. Evaluate tradeoffs.
4. Recommend a decision or next experiment.
5. Record decisions in `memory/decisions.md`.

## Project Review

1. Update `memory/projects.md`.
2. Identify blocked, stale, or high-risk work.
3. Define the next concrete action for each active project.
4. Move raw notes from `memory/inbox.md` into durable files.
